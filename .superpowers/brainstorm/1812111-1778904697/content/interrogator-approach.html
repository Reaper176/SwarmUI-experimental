Lodestone Image Interrogator Extension

Recommended self-contained shape for the first version.

Workflow
Generate Tab
selected/current image
→
Image Interrogator Panel
setup, preview, tag controls
→
Generate Prompt
append, replace, or copy tags

Implementation Approaches

A

Bundled Local Runner (Recommended)

Extension ships C# routes, UI assets, and a Python runner. A setup button installs dependencies and downloads the 5.27 GB model/vocab files before first use.

B

Comfy Backend Node

Integrates through Swarm's Comfy backend. Good for generation workflows, but more coupled to Comfy internals and less direct for a utility panel.

C

External Server Wrapper

Calls a separately running tagger UI/server. Fastest to prototype, but not self-contained and harder to share cleanly.

Panel Capabilities

Included in v1

  • Explicit setup button for model/dependency download.
  • Single-image interrogation.
  • Confidence threshold, max tags, and category filters.
  • Grouped rating/character/general/style tag views.
  • Send image from Generate to panel, send tags back to Generate.

Deferred

  • Batch interrogation.
  • Training dataset management.
  • Remote hosted inference.
